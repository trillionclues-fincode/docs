---
title: 'Get Dynamic Fields'
openapi: 'GET /dynamicfields/dynamic-fields-group/{processGroupCode}'
description: 'Retrieves a set of dynamic input fields required for a specific process (e.g., creating a transaction).'
---

Retrieves the structured set of dynamic fields (forms, documents, or additional data) required for a specific business process, identified by the `{processGroupCode}` in the URL.

This allows client applications to adapt their user interface to the requirements of the chosen transaction type or user state.
