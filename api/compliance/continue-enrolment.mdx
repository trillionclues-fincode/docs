---
title: "Continue Enrolment"
api: "GET https://remitjunction.fincode.software/api/v6/services/admin/compliance/continue-enrolment"
description: "Checks or triggers the next stage of external compliance screening for a customer, typically after a Customer Risk Assessment (CRA) form submission."
---

This endpoint is crucial for initiating or checking the status of required **external compliance screenings** (e.g., identity verification, sanctions checks) after a customer has submitted their initial forms or documents.

It confirms whether the external process has started and provides the necessary URL if user interaction is required.

<RequestExample>
          ```bash cURL GET        
  [https://remitjunction.fincode.software/api/v6/services/admin/compliance/continue-enrolment](https://remitjunction.fincode.software/api/v6/services/admin/compliance/continue-enrolment)
          ```
</RequestExample>

## Request Headers

This administrative endpoint requires a valid **Access Token** for authorization.

<ParamField header='X-Auth-Token' type='string' required>
  The **JWT Access Token** of a user with appropriate permissions (e.g.,
  `MANAGER` or `AGENT`).
</ParamField>

<ParamField header='platform' type='string' required default='fincode'>
        Platform identifier. Use `fincode`.
</ParamField>

<ParamField header='uuid' type='string' required default='200'>
        Unique request identifier.
</ParamField>

---

## Request Body

This endpoint typically requires the identifier of the customer whose enrolment status is being queried or advanced.

<ParamField body='customerCode' type='string' required>
  The unique identifier of the customer whose enrolment process needs to be
  continued or checked.
</ParamField>

---

## Response

The response confirms the status of the external screening process. If a link is required for the user to proceed, it will be provided in `externalVerificationUrl`.

<ResponseField name='status' type='string' required default='SUCCESS'>
  The outcome status of the request.
</ResponseField>

<ResponseField name='message' type='string' required>
  A descriptive message indicating the next steps or status.
</ResponseField>

<ResponseField name='externalScreeningEnrolled' type='boolean' required>
  **`true`** if the customer is successfully enrolled in, or has completed, the
  required external compliance screening.
</ResponseField>

<ResponseField name='externalVerificationUrl' type='string'>
  The URL the customer should be redirected to if external interaction (like
  identity verification) is required. This is **`null`** if no external
  interaction is needed.
</ResponseField>

<ResponseField name='customerId' type='string'>
  The ID of the customer involved in the enrolment process.
</ResponseField>

<ResponseField name='validityMinutes' type='number'>
  The number of minutes the provided `externalVerificationUrl` is valid for (if
  a URL is returned).
</ResponseField>

<ResponseField name='createdDate' type='string'>
  Timestamp indicating when the external screening process was initiated or last
  updated.
</ResponseField>

## Code Examples

<CodeGroup>

```bash cURL
curl -X GET 'https://remitjunction.fincode.software/api/v6/services/admin/compliance/continue-enrolment' \
--header 'X-Auth-Token: YOUR_AGENT_ACCESS_TOKEN' \
--header 'platform: fincode' \
--header 'uuid: 200' \
--header 'Content-Type: application/json' \
--data '{
    "customerCode": "CUST98765"
}'
```

```javascript Node.js
const axios = require("axios");

const BASE_URL =
  "[https://remitjunction.fincode.software/api/v6/services/admin/compliance](https://remitjunction.fincode.software/api/v6/services/admin/compliance)";

async function continueEnrolment(accessToken, customerCode) {
  try {
    const response = await axios.get(
      `${BASE_URL}/continue-enrolment`,
      { customerCode: customerCode },
      {
        headers: {
          "X-Auth-Token": accessToken,
          platform: "fincode",
          uuid: "200",
          "Content-Type": "application/json",
        },
      }
    );

    const data = response.data;
    console.log(`Enrolment Status: ${data.message}`);
    if (data.externalVerificationUrl) {
      console.log(
        `ACTION REQUIRED: Redirect customer to: ${data.externalVerificationUrl}`
      );
      console.log(`Link valid for: ${data.validityMinutes} minutes.`);
    } else if (data.externalScreeningEnrolled === true) {
      console.log(
        "External screening is complete or underway. Awaiting result callback."
      );
    } else {
      console.log("Waiting for customer document submission.");
    }

    return data;
  } catch (error) {
    console.error(
      "Enrolment check failed:",
      error.response?.data || error.message
    );
    throw error;
  }
}

// Usage Example
const accessToken = "YOUR_AGENT_ACCESS_TOKEN";
const customerCode = "CUST98765";
continueEnrolment(accessToken, customerCode);
```
