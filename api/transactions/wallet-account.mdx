---
title: 'Wallet Account List'
openapi: 'GET /walletmanagement/walletaccountlist'
description: 'Retrieves a list of all wallet accounts associated with the authenticated user.'
---

Retrieves a list of all digital wallet accounts (e.g., currency accounts) registered under the current user's profile. This list is essential for displaying balances or selecting a source account for transactions.

## Request Headers

This endpoint requires a valid **Access Token** for authorization to identify the user.

<ParamField header='X-Auth-Token' type='string' required>
  The **JWT Access Token** obtained from the `/login` or `/refresh-token`
  endpoint.
</ParamField>

<ParamField header='platform' type='string' required default='fincode'>
        Platform identifier. Use `fincode`.
</ParamField>

<ParamField header='uuid' type='string' required default='200'>
        Unique request identifier.
</ParamField>

## Code Examples

<CodeGroup>

```bash cURL
curl --location 'https://remitjunction.fincode.software/api/v6/services/walletmanagement/walletaccountlist' \
--header 'X-Auth-Token: YOUR_JWT_ACCESS_TOKEN' \
--header 'platform: fincode' \
--header 'uuid: 200'
```

```javascript Node.js
const axios = require('axios');

const BASE_URL =
  'https://remitjunction.fincode.software/api/v6/services/walletmanagement';

async function getWalletAccountList(accessToken) {
  try {
    const response = await axios.get(`${BASE_URL}/walletaccountlist`, {
      headers: {
        'X-Auth-Token': accessToken,
        platform: 'fincode',
        uuid: '200',
      },
    });

    const walletAccounts = response.data.data;

    console.log(`Wallets Found: ${walletAccounts.length}`);
    walletAccounts.forEach((wallet) => {
      console.log(
        `- ${wallet.currency} Wallet: ${wallet.availableBalance} ${wallet.currency} (ID: ${wallet.walletAccountId})`
      );
    });

    return walletAccounts;
  } catch (error) {
    console.error(
      'Failed to fetch wallet list:',
      error.response?.data || error.message
    );
    throw error;
  }
}

```
