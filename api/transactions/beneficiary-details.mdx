---
title: 'Beneficiary Details'
openapi: 'GET /usermanagement/view-beneficiery-details/{beneficiaryId}'
description: 'Retrieves the full profile and banking details for a specific saved beneficiary.'
---

Retrieves all stored details for a single beneficiary, identified by the unique `{beneficiaryId}`. This includes personal information and the associated payout account details.

## Request Headers

This endpoint requires a valid **Access Token** for authorization.

<ParamField header='X-Auth-Token' type='string' required>
  The **JWT Access Token** obtained from the `/login` or `/refresh-token`
  endpoint.
</ParamField>

<ParamField header='platform' type='string' required default='fincode'>
        Platform identifier. Use `fincode`.
</ParamField>

<ParamField header='uuid' type='string' required default='200'>
        Unique request identifier.
</ParamField>

## Path Parameters

<ParamField
  path='beneficiaryId'
  type='string'
  required
  default='6327c026-c646-4a9d-86b2-57731d867727'
>
  The unique ID of the beneficiary whose details are being requested.
</ParamField>

## Code Examples

<CodeGroup>

```bash cURL
curl --location 'https://remitjunction.fincode.software/api/v6/services/usermanagement/view-beneficiery-details/6327c026-c646-4a9d-86b2-57731d867727' \
--header 'X-Auth-Token: YOUR_JWT_ACCESS_TOKEN' \
--header 'platform: fincode' \
--header 'uuid: 200'
```

```javascript Node.js
const axios = require('axios');

const BASE_URL =
  'https://remitjunction.fincode.software/api/v6/services/usermanagement';

async function viewBeneficiaryDetails(accessToken, beneficiaryId) {
  const url = `${BASE_URL}/view-beneficiery-details/${beneficiaryId}`;
  try {
    const response = await axios.get(url, {
      headers: {
        'X-Auth-Token': accessToken,
        platform: 'fincode',
        uuid: '200',
      },
    });

    const beneficiary = response.data;

    console.log(
      `Fetched Beneficiary: ${beneficiary.firstName} ${beneficiary.lastName}`
    );
    console.log(`Receiving Country: ${beneficiary.countryCode}`);
    console.log(`Bank: ${beneficiary.bankDetails.bankName}`);
    console.log(`Account: ${beneficiary.bankDetails.accountNumber}`);

    return beneficiary;
  } catch (error) {
    console.error(
      'Failed to fetch beneficiary details:',
      error.response?.data || error.message
    );
    throw error;
  }
}

```
