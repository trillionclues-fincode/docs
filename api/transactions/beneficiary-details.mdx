---
title: 'Beneficiary Details'
api: 'GET https://remitjunction.fincode.software/api/v6/services/usermanagement/view-beneficiery-details/{beneficiaryId}'
description: 'Retrieves the full profile and banking details for a specific saved beneficiary.'
---

Retrieves all stored details for a single beneficiary, identified by the unique `{beneficiaryId}`. This includes personal information and the associated payout account details.

<RequestExample>
          ```bash cURL GET        
  [https://remitjunction.fincode.software/api/v6/services/usermanagement/view-beneficiery-details/6327c026-c646-4a9d-86b2-57731d867727](https://remitjunction.fincode.software/api/v6/services/usermanagement/view-beneficiery-details/6327c026-c646-4a9d-86b2-57731d867727)
          ```
</RequestExample>

## Request Headers

This endpoint requires a valid **Access Token** for authorization.

<ParamField header='X-Auth-Token' type='string' required>
  The **JWT Access Token** obtained from the `/login` or `/refresh-token`
  endpoint.
</ParamField>

<ParamField header='platform' type='string' required default='fincode'>
        Platform identifier. Use `fincode`.
</ParamField>

<ParamField header='uuid' type='string' required default='200'>
        Unique request identifier.
</ParamField>

## Path Parameters

<ParamField
  path='beneficiaryId'
  type='string'
  required
  default='6327c026-c646-4a9d-86b2-57731d867727'
>
  The unique ID of the beneficiary whose details are being requested.
</ParamField>

## Response

The response returns a comprehensive object containing the beneficiary's profile and banking information.

<ResponseField name='beneficiaryId' type='string'>
  Unique system ID of the beneficiary.
</ResponseField>

<ResponseField name='firstName' type='string'>
  Beneficiary's first name.
</ResponseField>

<ResponseField name='lastName' type='string'>
  Beneficiary's last name.
</ResponseField>

<ResponseField name='countryCode' type='string'>
  The country where the beneficiary resides (ISO 3166-1 alpha-3 code).
</ResponseField>

<ResponseField name='email' type='string'>
  Beneficiary's email address.
</ResponseField>

<ResponseField name='bankDetails' type='object'>
  The saved banking or payout information for the beneficiary.
  <Expandable title='bankDetails object'>
    <ResponseField name='accountName' type='string'>
      Name on the bank account.
    </ResponseField>
    <ResponseField name='accountNumber' type='string'>
      The full account number or ID.
    </ResponseField>
    <ResponseField name='bankName' type='string'>
      Name of the bank.
    </ResponseField>
    <ResponseField name='swiftBic' type='string'>
      SWIFT/BIC code (if applicable).
    </ResponseField>
    <ResponseField name='bankCode' type='string'>
      The country-specific bank code (e.g., routing number, sort code).
    </ResponseField>
    <ResponseField name='currencyCode' type='string'>
      The currency the account accepts (e.g., `NGN`).
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name='message' type='string'>
  Confirmation message from the server.
</ResponseField>

## Code Examples

<CodeGroup>

```bash cURL
curl --location 'https://remitjunction.fincode.software/api/v6/services/usermanagement/view-beneficiery-details/6327c026-c646-4a9d-86b2-57731d867727' \
--header 'X-Auth-Token: YOUR_JWT_ACCESS_TOKEN' \
--header 'platform: fincode' \
--header 'uuid: 200'
```

```javascript Node.js
const axios = require('axios');

const BASE_URL =
  'https://remitjunction.fincode.software/api/v6/services/usermanagement';

async function viewBeneficiaryDetails(accessToken, beneficiaryId) {
  const url = `${BASE_URL}/view-beneficiery-details/${beneficiaryId}`;
  try {
    const response = await axios.get(url, {
      headers: {
        'X-Auth-Token': accessToken,
        platform: 'fincode',
        uuid: '200',
      },
    });

    const beneficiary = response.data;

    console.log(
      `Fetched Beneficiary: ${beneficiary.firstName} ${beneficiary.lastName}`
    );
    console.log(`Receiving Country: ${beneficiary.countryCode}`);
    console.log(`Bank: ${beneficiary.bankDetails.bankName}`);
    console.log(`Account: ${beneficiary.bankDetails.accountNumber}`);

    return beneficiary;
  } catch (error) {
    console.error(
      'Failed to fetch beneficiary details:',
      error.response?.data || error.message
    );
    throw error;
  }
}

// Usage Example
const accessToken = 'YOUR_JWT_ACCESS_TOKEN';
const beneficiaryId = '6327c026-c646-4a9d-86b2-57731d867727';
viewBeneficiaryDetails(accessToken, beneficiaryId);
```
