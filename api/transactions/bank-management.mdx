These endpoints are used to retrieve supported banks, required fields, and to link bank accounts to beneficiaries.

<RequestExample>
  ```bash cURL GET
  [https://remitjunction.fincode.software/api/v6/services/bankmanagement/list-supported-banks-iso3/{country}](https://remitjunction.fincode.software/api/v6/services/bankmanagement/list-supported-banks-iso3/{country})
  ```
</RequestExample>

## List Supported Banks

Retrieves the list of supported banks for a specific country.

<Card title="Endpoint" icon="building-columns">
  `GET https://remitjunction.fincode.software/api/v6/services/bankmanagement/list-supported-banks-iso3/{country}`
</Card>

**Example Request:**
`GET .../list-supported-banks-iso3/NGA`

---

## Get Required Fields

Retrieves the specific bank fields required for a destination country (e.g., if Swift Code or Branch Code is needed).

<Card title="Endpoint" icon="list-check">
  `GET https://remitjunction.fincode.software/api/v6/services/bankmanagement/required-bank-fields/{country}`
</Card>

**Example Request:**
`GET .../required-bank-fields/NGA`

**Example Response:**
```json
{
    "requiredBankFields": {
        "bankNameRequired": true,
        "accountNumberRequired": true,
        "swiftCodeRequired": true,
        "country": "NGA",
        ...
    }
}
```

---

## Add Bank Details

Links a validated bank account to a beneficiary profile.

<Card title="Endpoint" icon="plus">
  `POST https://remitjunction.fincode.software/api/v6/services/bankmanagement/add-bank-details-key-value-pairs`
</Card>

**Request Body:**

```json
{
    "id": "beneficiary_uuid",
    "customerId": "customer_uuid",
    "accountOwnerCustomerCode": "RZ0LNK",
    "currencyCode": "NGN",
    "country": "NGA",
    "bankAccountMetaData": [
        {"key": "bankname", "value": "FCMB"},
        {"key": "bankaccountno", "value": "8883968718"},
        {"key": "country", "value": "NGA"},
        {"key": "accountholdername", "value": "VICTOR ADEBIYI ADEYINKA"}
    ]
}
```
