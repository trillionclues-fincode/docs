---
title: 'List Mobile Operators'
api: 'GET https://remitjunction.fincode.software/api/v6/services/admin/bank/config/find-all-mobile-operator/{country}'
description: 'Retrieves the list of supported mobile money operators for a country.'
---

This endpoint returns the available mobile money operators (e.g., MTN, Orange, Airtel) for a specific country. The operator code is required for validating mobile numbers and creating transactions.

<RequestExample>
  ```bash cURL GET
  [https://remitjunction.fincode.software/api/v6/services/admin/bank/config/find-all-mobile-operator/{country}](https://remitjunction.fincode.software/api/v6/services/admin/bank/config/find-all-mobile-operator/{country})
  ```
</RequestExample>

## Request Headers

<ParamField header='X-Auth-Token' type='string' required>
  The **JWT Access Token** of the authenticated user.
</ParamField>

<ParamField header='platform' type='string' required default='fincode'>
  Platform identifier. Use `fincode`.
</ParamField>

<ParamField header='uuid' type='string' required default='200'>
  Unique request identifier.
</ParamField>

## Path Parameters

<ParamField path='country' type='string' required>
  ISO 3166-1 alpha-3 country code (e.g., `GIN`).
</ParamField>

## Code Example

```bash cURL
curl -X GET 'https://remitjunction.fincode.software/api/v6/services/admin/bank/config/find-all-mobile-operator/GIN' \
--header 'X-Auth-Token: YOUR_TOKEN' \
--header 'platform: fincode' \
--header 'uuid: 200'
```
