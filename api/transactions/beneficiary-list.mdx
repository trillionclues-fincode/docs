---
title: 'Find All Beneficiary'
api: 'GET https://remitjunction.fincode.software/api/v6/services/usermanagement/find-all-beneficiery'
description: 'Retrieves a list of all saved beneficiaries associated with the authenticated customer or managed by the agent.'
---

This endpoint fetches all saved recipient profiles (**beneficiaries**) linked to the authenticated user's account. This list is typically displayed when a user initiates a new transaction to select a pre-saved recipient.

<RequestExample>
          ```bash cURL GET        
  [https://remitjunction.fincode.software/api/v6/services/usermanagement/find-all-beneficiery](https://remitjunction.fincode.software/api/v6/services/usermanagement/find-all-beneficiery)
          ```
</RequestExample>

## Request Headers

This endpoint requires a valid **Access Token** for authorization to identify the user and their associated beneficiaries.

<ParamField header='X-Auth-Token' type='string' required>
  The **JWT Access Token** obtained from the `/login` or `/refresh-token`
  endpoint.
</ParamField>

<ParamField header='platform' type='string' required default='fincode'>
        Platform identifier. Use `fincode`.
</ParamField>

<ParamField header='uuid' type='string' required default='200'>
        Unique request identifier.
</ParamField>

## Response

The response returns an array containing details for each saved beneficiary.

<ResponseField name='status' type='string' default='SUCCESS'>
  Overall status of the API request.
</ResponseField>

<ResponseField name='data' type='array'>
  A list of the user's saved beneficiaries.
  <Expandable title='data array item (Beneficiary Details)'>
    <ResponseField name='beneficiaryId' type='string'>
      Unique system ID for the beneficiary.
    </ResponseField>
    <ResponseField name='name' type='string'>
      Full name of the recipient.
    </ResponseField>
    <ResponseField name='country' type='string'>
      The country where the beneficiary resides or receives funds (ISO 3166-1
      alpha-3 code).
    </ResponseField>
    <ResponseField name='payoutMethod' type='string'>
      The preferred method for payment (e.g., `BANK_ACCOUNT`, `CASH_PICKUP`).
    </ResponseField>
    <ResponseField name='accountNumber' type='string'>
      The recipient's bank account number or identifier (may be masked).
    </ResponseField>
    <ResponseField name='bankName' type='string'>
      Name of the recipient's bank or payout institution.
    </ResponseField>
    <ResponseField name='currency' type='string'>
      The currency the beneficiary usually receives.
    </ResponseField>
  </Expandable>
</ResponseField>

## Code Examples

<CodeGroup>

```bash cURL
curl --location 'https://remitjunction.fincode.software/api/v6/services/usermanagement/find-all-beneficiery' \
--header 'X-Auth-Token: YOUR_JWT_ACCESS_TOKEN' \
--header 'platform: fincode' \
--header 'uuid: 200'
```

```javascript Node.js
const axios = require('axios');

const BASE_URL =
  'https://remitjunction.fincode.software/api/v6/services/usermanagement';

async function findAllBeneficiaries(accessToken) {
  try {
    const response = await axios.get(`${BASE_URL}/find-all-beneficiery`, {
      headers: {
        'X-Auth-Token': accessToken,
        platform: 'fincode',
        uuid: '200',
      },
    });

    const beneficiaries = response.data.data;

    console.log(`Beneficiaries Found: ${beneficiaries.length}`);
    beneficiaries.forEach((b) => {
      console.log(`- ${b.name} (${b.country}) - Account: ${b.accountNumber}`);
    });

    return beneficiaries;
  } catch (error) {
    console.error(
      'Failed to fetch beneficiaries:',
      error.response?.data || error.message
    );
    throw error;
  }
}

// Usage Example
const accessToken = 'YOUR_JWT_ACCESS_TOKEN';
findAllBeneficiaries(accessToken);
```

</CodeGroup>
