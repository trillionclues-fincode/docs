---
title: 'Active Countries'
api: 'GET https://remitjunction.fincode.software/api/v6/services/utilitiesmanagement/active-countries'
description: "Retrieves the systems's saved country list."
---

Retrieves the current list of active countries stored in the system with their various properties like country name and iso3Codes. This endpoint helps client applications for where users country is required.

<RequestExample>
          ```bash cURL GET        
  [https://remitjunction.fincode.software/api/v6/services/utilitiesmanagement/active-countries](https://remitjunction.fincode.software/api/v6/services/utilitiesmanagement/active-countries)
          ```
</RequestExample>

## Request Headers

This endpoint is typically **public** and **does not require an Access Token** for authorization, relying on the organization domain provided in the URL.

<ParamField header='platform' type='string' required default='fincode'>
        Platform identifier. Use `fincode`.
</ParamField>

<ParamField header='uuid' type='string' required default='200'>
        Unique request identifier.
</ParamField>

## Response

The response returns an object detailing all the active countries of the world with their different iso3Codes.

<ResponseField name='iso3Code' type='string'>
    Unique identifier for the country record.
</ResponseField>

<ResponseField name='common_name' type='string'>
    The country's common name.
</ResponseField>

## Code Examples

<CodeGroup>

```bash cURL
curl --location 'https://remitjunction.fincode.software/api/v6/services/utilitiesmanagement/active-countries' \
--header 'platform: fincode' \
--header 'uuid: 200'
```

```javascript Node.js
const axios = require('axios');

const BASE_URL =
  'https://remitjunction.fincode.software/api/v6/services/utilitiesmanagement';

async function viewCountryList() {
  try {
    const response = await axios.get(`${BASE_URL}/active-countries`, {
      headers: {
        platform: 'fincode',
        uuid: '200',
      },
    });
    return response.data;
  } catch (error) {
    console.error(
      'Failed to fetch country list:',
      error.response?.data || error.message
    );
    throw error;
  }
}

viewCountryList();
```

</CodeGroup>
