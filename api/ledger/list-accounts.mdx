---
title: 'List Accounts'
openapi: 'GET /account-maintenance/list-accounts/{account-owner-id}'
description: 'Retrieves a list of all accounts owned by a specific account owner (customer).'
---

Fetches all accounts associated with a specific account owner, allowing customers to view all their accounts in one place.

## Request Headers

<ParamField header='X-Auth-Token' type='string' required>
  The **JWT Access Token** obtained from the `/login` or `/refresh-token` endpoint.
</ParamField>

<ParamField header='x-idempotency-key' type='string' required>
  Unique idempotency key for the request to prevent duplicate processing.
</ParamField>

<ParamField header='x-fapi-auth-date' type='string' required>
  The date and time at which the request was initiated (ISO 8601 format).
</ParamField>

<ParamField header='x-fapi-customer-ip-address' type='string' required>
  The IP address of the customer making the request.
</ParamField>

<ParamField header='x-fapi-interaction-id' type='string' required>
  Unique identifier for the interaction/session.
</ParamField>

## Path Parameters

<ParamField path='account-owner-id' type='string' required>
  The unique identifier (UUID) of the account owner (customer) whose accounts to retrieve.
</ParamField>

```bash cURL
curl --location 'https://{domain}.fincode.software/api/v6/services/account-maintenance/list-accounts/123e4567-e89b-12d3-a456-426614174000' \

--header 'x-fapi-customer-ip-address: 192.168.1.1' \
--header 'x-fapi-interaction-id: interaction-12345'
```

```javascript Node.js
const axios = require('axios');

const BASE_URL = 'https://{domain}.fincode.software/api/v6/services/account-maintenance';

async function listAccounts(accessToken, accountOwnerId) {
  try {
    const response = await axios.get(
      `${BASE_URL}/list-accounts/${accountOwnerId}`,
      {
        headers: {
          'X-Auth-Token': accessToken,
          'x-idempotency-key': `key-${Date.now()}`,
          'x-fapi-auth-date': new Date().toISOString(),
          'x-fapi-customer-ip-address': '192.168.1.1',
          'x-fapi-interaction-id': `interaction-${Date.now()}`,
        },
      }
    );

    console.log(`Found ${response.data.data.length} accounts:`);
    response.data.data.forEach((account) => {
      console.log(
        `${account.accountNumber}: ${account.accountName} ` +
        `(${account.currencyCode}) - Balance: ${account.accountBalance}`
      );
    });

    return response.data.data;
  } catch (error) {
    console.error(
      'Failed to list accounts:',
      error.response?.data || error.message
    );
    throw error;
  }
}

```
