---
title: 'Create Transaction PIN'
api: 'POST https://remitjunction.fincode.software/api/v1/services/user/create-transaction-pin'
description: 'Creates a transaction PIN for the customer to authorize loan-related transactions.'
---

Allows customers to create a transaction PIN that will be used to authorize loan repayments and other sensitive transactions.

<RequestExample>
  ```bash cURL POST
  [https://remitjunction.fincode.software/api/v1/services/user/create-transaction-pin](https://remitjunction.fincode.software/api/v1/services/user/create-transaction-pin)
  ```
</RequestExample>

## Request Headers

<ParamField header='X-Auth-Token' type='string' required>
  The **JWT Access Token** obtained from the `/login` or `/refresh-token` endpoint.
</ParamField>

<ParamField header='x-idempotency-key' type='string' required>
  Unique idempotency key for the request to prevent duplicate processing.
</ParamField>

<ParamField header='x-fapi-auth-date' type='string' required>
  The date and time at which the request was initiated (ISO 8601 format).
</ParamField>

<ParamField header='x-fapi-customer-ip-address' type='string' required>
  The IP address of the customer making the request.
</ParamField>

<ParamField header='x-fapi-interaction-id' type='string' required>
  Unique identifier for the interaction/session.
</ParamField>

<ParamField header='Content-Type' type='string' required default='application/json'>
  Must be `application/json`.
</ParamField>

## Request Body

<ParamField body='customerId' type='string' required>
  The unique identifier (UUID) of the customer creating the transaction PIN.
</ParamField>

<ParamField body='transactionPin' type='string' required>
  The transaction PIN to be created. Should meet security requirements (typically 4-6 digits).
</ParamField>

<ParamField body='confirmTransactionPin' type='string' required>
  Confirmation of the transaction PIN to ensure accuracy.
</ParamField>

## Response

Returns confirmation that the transaction PIN was created successfully.

<ResponseField name='status' type='string' default='SUCCESS'>
  Overall status of the API request.
</ResponseField>

<ResponseField name='data' type='object'>
  Confirmation details.
  <Expandable title='data object'>
    <ResponseField name='message' type='string'>
      Confirmation message indicating successful PIN creation.
    </ResponseField>
  </Expandable>
</ResponseField>

```javascript Node.js
const axios = require('axios');

const BASE_URL = 'https://remitjunction.fincode.software/api/v1/services/user';

async function createTransactionPin(accessToken, pinData) {
  try {
    const response = await axios.post(
      `${BASE_URL}/create-transaction-pin`,
      {
        customerId: pinData.customerId,
        transactionPin: pinData.transactionPin,
        confirmTransactionPin: pinData.confirmTransactionPin,
      },
      {
        headers: {
          'X-Auth-Token': accessToken,
          'x-idempotency-key': `key-${Date.now()}`,
          'x-fapi-auth-date': new Date().toISOString(),
          'x-fapi-customer-ip-address': '192.168.1.1',
          'x-fapi-interaction-id': `interaction-${Date.now()}`,
          'Content-Type': 'application/json',
        },
      }
    );

    console.log('Transaction PIN created successfully!');
    return response.data.data;
  } catch (error) {
    console.error(
      'Failed to create transaction PIN:',
      error.response?.data || error.message
    );
    throw error;
  }
}

// Usage Example
const accessToken = 'YOUR_JWT_ACCESS_TOKEN';
createTransactionPin(accessToken, {
  customerId: '123e4567-e89b-12d3-a456-426614174000',
  transactionPin: '1234',
  confirmTransactionPin: '1234',
});
```
