---
title: 'Available Banks'
api: 'GET https://finlend.fincode.software/api/v1/services/user/get-available-banks/'
description: 'Retrieves a list of supported banks for the system.'
---

Returns a list of available banks (e.g., Nigerian banks) that can be used when adding a new customer bank account.

<RequestExample>
  ```bash cURL
  curl --request GET 'https://finlend.fincode.software/api/v1/services/user/get-available-banks/' \
  --header 'Authorization: Bearer <token>'
  ```
</RequestExample>
