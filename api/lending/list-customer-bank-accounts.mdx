---
title: 'List Bank Accounts'
api: 'GET https://finlend.fincode.software/api/v1/services/user/list-customer-bank-accounts/{customerId}'
description: 'Retrieves a list of saved bank accounts for the customer.'
---

Returns the list of bank accounts associated with the customer, which can be used for withdrawing disbursed loans.

<RequestExample>
  ```bash cURL
  curl --request GET 'https://finlend.fincode.software/api/v1/services/user/list-customer-bank-accounts/8ecca811-607f-44e1-a57e-cd856c50437d' \
  --header 'Authorization: Bearer <token>'
  ```
</RequestExample>
