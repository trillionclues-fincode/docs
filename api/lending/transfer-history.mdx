---
title: 'Transfer History'
api: 'GET https://finlend.fincode.software/api/v1/services/user/transfer-history/{customerId}'
description: 'Retrieves the history of transfers (withdrawals) made by the customer.'
---

Returns a list of transfers initiated by the customer, typically withdrawals from their loan wallet to their bank account.

<RequestExample>
  ```bash cURL
  curl --request GET 'https://finlend.fincode.software/api/v1/services/user/transfer-history/8ecca811-607f-44e1-a57e-cd856c50437d' \
  --header 'Authorization: Bearer <token>'
  ```
</RequestExample>

## Request Headers

<ParamField header='X-Auth-Token' type='string' required>
  The **JWT Access Token** obtained from the `/login` or `/refresh-token` endpoint.
</ParamField>

<ParamField header='x-idempotency-key' type='string' required>
  Unique idempotency key for the request to prevent duplicate processing.
</ParamField>

<ParamField header='x-fapi-auth-date' type='string' required>
  The date and time at which the request was initiated (ISO 8601 format).
</ParamField>

<ParamField header='x-fapi-customer-ip-address' type='string' required>
  The IP address of the customer making the request.
</ParamField>

<ParamField header='x-fapi-interaction-id' type='string' required>
  Unique identifier for the interaction/session.
</ParamField>

## Path Parameters

<ParamField path="customerId" type="string" required>
  The unique identifier of the customer.
</ParamField>

## Response

<ResponseField name="status" type="string">
  The status of the request.
</ResponseField>

<ResponseField name="data" type="array">
  List of transfers.
  <Expandable title="Transfer Object">
    <ResponseField name="reference" type="string">
      Transaction reference ID.
    </ResponseField>
    <ResponseField name="amount" type="number">
      Amount transferred.
    </ResponseField>
    <ResponseField name="status" type="string">
      Status of the transfer (e.g., SUCCESSFUL, PENDING).
    </ResponseField>
    <ResponseField name="date" type="string">
      Date of the transfer.
    </ResponseField>
  </Expandable>
</ResponseField>
