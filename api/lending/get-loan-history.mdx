---
title: 'Loan History'
api: 'GET https://finlend.fincode.software/api/v1/services/user/get-loan-history/{customerId}'
description: 'Retrieves the loan application history for a customer.'
---

Returns a list of all loan applications made by the customer, including their current status.

<RequestExample>
  ```bash cURL
  curl --request GET 'https://finlend.fincode.software/api/v1/services/user/get-loan-history/8ecca811-607f-44e1-a57e-cd856c50437d' \
  --header 'Authorization: Bearer <token>'
  ```
</RequestExample>
