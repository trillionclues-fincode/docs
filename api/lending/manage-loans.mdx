---
title: 'Manage Loans'
api: 'GET https://finlend.fincode.software/api/v1/services/user/manage-loans/{customerId}'
description: 'Retrieves active loans that are due for repayment.'
---

Returns details of loans that are currently active and may require repayment management.

<RequestExample>
  ```bash cURL
  curl --request GET 'https://finlend.fincode.software/api/v1/services/user/manage-loans/8ecca811-607f-44e1-a57e-cd856c50437d' \
  --header 'Authorization: Bearer <token>'
  ```
</RequestExample>
