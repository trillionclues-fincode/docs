---
title: 'Eligibility Questions'
api: 'GET https://finlend.fincode.software/api/v1/services/user/loan-eligibility-questions/{productId}'
description: 'Retrieves dynamic eligibility questions for a specific loan product.'
---

Returns a list of questions that the customer must answer to determine their eligibility for the selected loan product.

<RequestExample>
  ```bash cURL
  curl --request GET 'https://finlend.fincode.software/api/v1/services/user/loan-eligibility-questions/11d936b5-f6db-45d6-b35e-2ce24791997f' \
  --header 'Authorization: Bearer <token>'
  ```
</RequestExample>
