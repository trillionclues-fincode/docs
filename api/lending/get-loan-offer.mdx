---
title: 'Get Loan Offer'
api: 'GET https://finlend.fincode.software/api/v1/services/user/get-loan-offer'
description: 'Retrieves the current loan offer for the customer.'
---

Returns the loan offer details made to the customer after the underwriting process. The offer may differ from the requested amount.

<RequestExample>
  ```bash cURL
  curl --request GET 'https://finlend.fincode.software/api/v1/services/user/get-loan-offer' \
  --header 'Authorization: Bearer <token>'
  ```
</RequestExample>

## Request Headers

<ParamField header='X-Auth-Token' type='string' required>
  The **JWT Access Token** obtained from the `/login` or `/refresh-token` endpoint.
</ParamField>

<ParamField header='x-idempotency-key' type='string' required>
  Unique idempotency key for the request to prevent duplicate processing.
</ParamField>

<ParamField header='x-fapi-auth-date' type='string' required>
  The date and time at which the request was initiated (ISO 8601 format).
</ParamField>

<ParamField header='x-fapi-customer-ip-address' type='string' required>
  The IP address of the customer making the request.
</ParamField>

<ParamField header='x-fapi-interaction-id' type='string' required>
  Unique identifier for the interaction/session.
</ParamField>

## Response

Returns a list of loan offers available to the customer.

<ResponseField name='status' type='string' default='SUCCESS'>
  Overall status of the API request.
</ResponseField>

<ResponseField name='data' type='array'>
  Array of loan offer objects.
  <Expandable title='data array item'>
    <ResponseField name='loanApplicationId' type='string'>
      Unique identifier for the loan application.
    </ResponseField>
    <ResponseField name='loanProductName' type='string'>
      Name of the loan product.
    </ResponseField>
    <ResponseField name='offeredAmount' type='number'>
      The loan amount being offered.
    </ResponseField>
    <ResponseField name='interestRate' type='number'>
      Interest rate for the loan offer.
    </ResponseField>
    <ResponseField name='tenure' type='number'>
      Loan tenure in months.
    </ResponseField>
    <ResponseField name='monthlyPayment' type='number'>
      Estimated monthly payment amount.
    </ResponseField>
    <ResponseField name='totalAmountPayable' type='number'>
      Total amount to be repaid over the loan tenure.
    </ResponseField>
    <ResponseField name='offerExpiryDate' type='string'>
      Date when the offer expires.
    </ResponseField>
    <ResponseField name='status' type='string'>
      Status of the offer (e.g., `OFFER_SENT`, `PENDING_ACCEPTANCE`).
    </ResponseField>
  </Expandable>
</ResponseField>
