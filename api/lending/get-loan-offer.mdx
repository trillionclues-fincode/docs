---
title: 'Get Loan Offer'
api: 'GET https://remitjunction.fincode.software/api/v1/services/user/get-loan-offer'
description: 'Retrieves all loan offers available to the authenticated customer.'
---

Fetches a list of all loan offers that have been sent to the customer and are awaiting their response.

<RequestExample>
  ```bash cURL GET
  [https://remitjunction.fincode.software/api/v1/services/user/get-loan-offer](https://remitjunction.fincode.software/api/v1/services/user/get-loan-offer)
  ```
</RequestExample>

## Request Headers

<ParamField header='X-Auth-Token' type='string' required>
  The **JWT Access Token** obtained from the `/login` or `/refresh-token` endpoint.
</ParamField>

<ParamField header='x-idempotency-key' type='string' required>
  Unique idempotency key for the request to prevent duplicate processing.
</ParamField>

<ParamField header='x-fapi-auth-date' type='string' required>
  The date and time at which the request was initiated (ISO 8601 format).
</ParamField>

<ParamField header='x-fapi-customer-ip-address' type='string' required>
  The IP address of the customer making the request.
</ParamField>

<ParamField header='x-fapi-interaction-id' type='string' required>
  Unique identifier for the interaction/session.
</ParamField>

## Response

Returns a list of loan offers available to the customer.

<ResponseField name='status' type='string' default='SUCCESS'>
  Overall status of the API request.
</ResponseField>

<ResponseField name='data' type='array'>
  Array of loan offer objects.
  <Expandable title='data array item'>
    <ResponseField name='loanApplicationId' type='string'>
      Unique identifier for the loan application.
    </ResponseField>
    <ResponseField name='loanProductName' type='string'>
      Name of the loan product.
    </ResponseField>
    <ResponseField name='offeredAmount' type='number'>
      The loan amount being offered.
    </ResponseField>
    <ResponseField name='interestRate' type='number'>
      Interest rate for the loan offer.
    </ResponseField>
    <ResponseField name='tenure' type='number'>
      Loan tenure in months.
    </ResponseField>
    <ResponseField name='monthlyPayment' type='number'>
      Estimated monthly payment amount.
    </ResponseField>
    <ResponseField name='totalAmountPayable' type='number'>
      Total amount to be repaid over the loan tenure.
    </ResponseField>
    <ResponseField name='offerExpiryDate' type='string'>
      Date when the offer expires.
    </ResponseField>
    <ResponseField name='status' type='string'>
      Status of the offer (e.g., `OFFER_SENT`, `PENDING_ACCEPTANCE`).
    </ResponseField>
  </Expandable>
</ResponseField>

## Code Examples

<CodeGroup>

```bash cURL
curl --location 'https://remitjunction.fincode.software/api/v1/services/user/get-loan-offer' \
--header 'X-Auth-Token: YOUR_JWT_ACCESS_TOKEN' \
--header 'x-idempotency-key: unique-key-12345' \
--header 'x-fapi-auth-date: 2024-01-15T10:30:00Z' \
--header 'x-fapi-customer-ip-address: 192.168.1.1' \
--header 'x-fapi-interaction-id: interaction-12345'
```

```javascript Node.js
const axios = require('axios');

const BASE_URL = 'https://remitjunction.fincode.software/api/v1/services/user';

async function getLoanOffer(accessToken) {
  try {
    const response = await axios.get(
      `${BASE_URL}/get-loan-offer`,
      {
        headers: {
          'X-Auth-Token': accessToken,
          'x-idempotency-key': `key-${Date.now()}`,
          'x-fapi-auth-date': new Date().toISOString(),
          'x-fapi-customer-ip-address': '192.168.1.1',
          'x-fapi-interaction-id': `interaction-${Date.now()}`,
        },
      }
    );

    const offers = response.data.data;
    console.log(`Found ${offers.length} loan offers`);
    
    offers.forEach((offer) => {
      console.log(`- ${offer.loanApplicationId}: ${offer.offeredAmount} at ${offer.interestRate}%`);
    });

    return offers;
  } catch (error) {
    console.error(
      'Failed to fetch loan offers:',
      error.response?.data || error.message
    );
    throw error;
  }
}

// Usage Example
const accessToken = 'YOUR_JWT_ACCESS_TOKEN';
getLoanOffer(accessToken);
```
