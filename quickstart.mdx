# Quick Start

Get started with FinCode's multitenant platform in three simple steps. This guide will walk you through environment setup, authentication, and processing your first transaction.

## Step 1: Environment Setup & Authentication

<AccordionGroup>
  <Accordion icon="key" title="Get Your API Credentials">
    1. **Access the Dashboard**: Go to [app.fincode.com](https://app.fincode.com) and create your account
    2. **Create Your First Tenant**: Set up your tenant profile and configuration
    3. **Generate API Keys**: Navigate to Settings → API Keys and create your sandbox keys
    4. **Download SDK** (Optional): Get our client libraries for faster integration

    <Tip>Keep your API keys secure! Never commit them to version control.</Tip>

  </Accordion>

  <Accordion icon="gear" title="Configure Your Environment">
    ### Sandbox vs Production
    - **Sandbox**: `https://api-sandbox.fincode.com` - For testing with simulated transactions
    - **Production**: `https://api.fincode.com` - For live transactions

    ### Environment Variables
    ```bash
    # Add to your environment configuration
    FINCODE_API_KEY=sk_sandbox_your_key_here
    FINCODE_BASE_URL=https://api-sandbox.fincode.com
    FINCODE_TENANT_ID=your_tenant_id
    ```

    <Warning>Always test thoroughly in sandbox before moving to production.</Warning>

  </Accordion>
</AccordionGroup>

## Step 2: Make Your First API Call

<AccordionGroup>
  <Accordion icon="terminal" title="Test Authentication">
    Verify your setup by making a simple API call to check your tenant status:

    ```bash
    curl -X GET "https://api-sandbox.fincode.com/v1/tenants/me" \
      -H "X-Tenant-ID: $FINCODE_TENANT_ID" \
      -H "Authorization: Bearer $FINCODE_API_KEY"
    ```

    **Expected Response:**
    ```json
    {
      "tenant_id": "tenant_123",
      "name": "Your Tenant Name",
      "status": "active",
      "products": ["remittance", "lending"],
      "created_at": "2024-01-15T10:30:00Z"
    }
    ```

    <Tip>Use this endpoint to verify your authentication and tenant configuration.</Tip>

  </Accordion>

  <Accordion icon="code" title="SDK Installation (Optional)">
    ### JavaScript/Node.js
    ```bash
    npm install @fincode/sdk
    ```

    ```javascript
    import { FinCode } from '@fincode/sdk';

    const client = new FinCode({
      apiKey: process.env.FINCODE_API_KEY,
      baseURL: process.env.FINCODE_BASE_URL,
      tenantId: process.env.FINCODE_TENANT_ID
    });

    // Test connection
    const tenant = await client.tenants.getMe();
    ```

    ### Python
    ```bash
    pip install fincode-python
    ```

    ```python
    from fincode import FinCode

    client = FinCode(
        api_key=os.getenv('FINCODE_API_KEY'),
        base_url=os.getenv('FINCODE_BASE_URL'),
        tenant_id=os.getenv('FINCODE_TENANT_ID')
    )

    tenant = client.tenants.get_me()
    ```

  </Accordion>
</AccordionGroup>
